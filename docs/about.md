# 关于本站

本站由 GitHub Actions 定时运行，不依赖个人电脑在线。

当前数据源：

- OpenJDK GitHub 仓库中的 Pull Request
- PR 描述、变更文件、Issue 评论和 Review
- 用户手工维护的贡献者—厂商映射

AI 只负责基于已采集事实生成技术观察。PR 编号、作者、状态、模块、架构和统计数据均由程序生成；AI 调用失败时，基础日报仍会正常发布。

## 准确性说明

“当日活动”表示报告日期内发生过创建、合并、关闭、评论、Review 或其他更新时间变化的 PR。GitHub 的搜索与事件接口并不能覆盖所有内部状态变化，因此它适合作为社区动态简报，而不是审计记录。
